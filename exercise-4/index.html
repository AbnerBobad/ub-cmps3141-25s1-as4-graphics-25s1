Exercise 4: Evaluating Learnability

Vector Graphics Editor Experience

Background

Editor Used: Figma (second-time user)

Previous Experience: This is my second time using Figma. While I have some basic familiarity with the interface, I still do not fully understand what all the different features and tools do.

Time Spent: Approximately 2-3 hours to complete the business card design and explore the tools.

Learning Process Screenshots

Pain Points and Struggles

  1. Finding the right tools: The toolbar icons were not immediately obvious. I had to hover over each one to understand their purpose. The shortcut keys (like R for rectangle, T for text) helped once I discovered them.
  2. Understanding frames vs groups: I was confused about when to use frames versus groups. I had to Google the difference and learned that frames are better for layout containers.
  3. Importing custom fonts: Installing the Museo font required downloading the Figma desktop app. This was not obvious from the web interface.
  4. Exporting as SVG: Finding the export options took some time. I initially looked in the File menu but later found it in the right sidebar under Design.
  5. Converting text to outlines: This was tricky to find. I had to Google how to "flatten" text in Figma and learned about the "Outline Stroke" and "Flatten" commands.

What I Had to Google

  • How to import SVG in Figma
  • Difference between frames and groups in Figma
  • How to install custom fonts in Figma
  • How to convert text to outlines in Figma
  • How to export as SVG in Figma
  • How to set exact dimensions for shapes

Learnability Assessment

Could the app have been more learnable?

Yes, I believe Figma could improve learnability in several ways:

  • Better onboarding: A quick interactive tutorial for first-time users would help introduce the basic concepts like frames, layers, and the component system.
  • Clearer tool labels: Having text labels alongside icons (or as a toggle option) would make the toolbar more discoverable.
  • Contextual help: Showing hints when hovering over panels would explain what each section does.
  • Search in properties: A search function in the right sidebar would help find specific settings faster.

However, Figma does have good documentation online, and the community tutorials were very helpful. Once I learned the basics, the interface became more intuitive.

Positive Aspects

  • The auto-save feature prevented any loss of work
  • Real-time collaboration features are impressive
  • The zoom and pan controls are smooth and responsive
  • Smart guides for alignment made precise positioning easy
  • The color picker and gradient tools were intuitive
