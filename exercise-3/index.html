Exercise 3: Evaluating Graphics

Design Principles Analysis

Contrast | Proximity

Contrast (Group A)

Contrast is the difference between elements that makes them distinguishable from one another. It helps create visual hierarchy and guides the viewer's attention to the most important information.

Improvement: To fix this flyer, I would add a dark semi-transparent overlay behind the text, increase the font weight, and use white or bright yellow text to create clear separation from the background image.

This poster follows the contrast principle effectively by using bold white typography against a dark background, making the message instantly readable and creating a strong visual impact.

Proximity (Group B)

Proximity is the principle of grouping related items together. Items that are close to each other are perceived as related, while items that are far apart are seen as separate groups.

Improvement: To improve this menu, I would group each dish name with its price and description into compact blocks, add clear section dividers between categories, and reduce the spacing between related elements.

This menu follows the proximity principle by keeping dish names, prices, and descriptions together in clear blocks. The organized grouping makes it easy for customers to scan and find what they want quickly.
